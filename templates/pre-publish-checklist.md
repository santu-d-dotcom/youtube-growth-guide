# Pre-Publish Checklist

Use this checklist before publishing every video to ensure optimal performance.

## Content Quality

- [ ] Provides clear value to audience
- [ ] Unique or offers new perspective
- [ ] Information is accurate and well-researched
- [ ] Production quality is professional
- [ ] Content matches title/thumbnail promise

---

## Title Optimization

- [ ] Includes primary keyword
- [ ] Creates curiosity or promises benefit
- [ ] 50-60 characters optimal length
- [ ] Clear and appealing
- [ ] Uses power words (if appropriate)
- [ ] Mobile-friendly (not truncated)

---

## Thumbnail Optimization

- [ ] High contrast and vibrant colors
- [ ] Readable text (if used) - minimum 60pt font
- [ ] Creates curiosity
- [ ] Mobile-optimized (readable at small size)
- [ ] Brand consistent
- [ ] Stands out in feed
- [ ] Accurately represents content

---

## SEO Optimization

- [ ] Comprehensive description (200+ words minimum)
- [ ] Keywords included naturally
- [ ] Primary keyword in first 125 characters of description
- [ ] Timestamps added (if applicable)
- [ ] Relevant tags (10-15 tags)
- [ ] Mix of broad and specific tags
- [ ] Playlist added (if applicable)
- [ ] Cards and end screens added
- [ ] Community tab post (if applicable)

---

## Audience Alignment

- [ ] Matches audience expectations
- [ ] Addresses audience needs
- [ ] Appropriate difficulty level
- [ ] Relevant to niche
- [ ] Consistent with channel theme

---

## Engagement Optimization

- [ ] Strong opening hook (first 15 seconds)
- [ ] Clear call-to-action (subscribe, like, comment)
- [ ] Multiple CTAs throughout video
- [ ] Encourages comments (asks questions)
- [ ] Share-worthy moments included
- [ ] End screen with related videos

---

## Competitive Positioning

- [ ] Different from competitors
- [ ] Clear unique value proposition
- [ ] Right timing (trend alignment if applicable)
- [ ] Fills content gap or improves on existing content

---

## Production Value

- [ ] High video quality (1080p minimum, 4K if possible)
- [ ] Clear audio (no background noise)
- [ ] Good audio levels (not too quiet/loud)
- [ ] Professional editing
- [ ] Smooth transitions
- [ ] Good pacing
- [ ] No obvious mistakes
- [ ] Consistent branding

---

## Technical Checks

- [ ] Video uploaded successfully
- [ ] Thumbnail uploaded and selected
- [ ] Title entered correctly
- [ ] Description formatted properly
- [ ] Tags added
- [ ] Playlist selected (if applicable)
- [ ] Category selected
- [ ] Language and captions set
- [ ] Visibility set correctly
- [ ] Cards added (if applicable)
- [ ] End screen added (if applicable)
- [ ] Community tab post created (if applicable)

---

## Final Review

- [ ] Content scored using scoring template
- [ ] Score above 60 (Good) or improvements made
- [ ] All checklist items completed
- [ ] Ready to publish

---

## Post-Publish

- [ ] Share on social media
- [ ] Post in community tab
- [ ] Respond to initial comments
- [ ] Monitor performance in first 24 hours
- [ ] Track CTR and engagement
