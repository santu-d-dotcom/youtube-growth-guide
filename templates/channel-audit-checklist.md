# Channel Audit Checklist

Use this checklist to conduct a comprehensive channel audit.

## Phase 1: Data Collection

### Analytics Export
- [ ] Exported YouTube Analytics data (last 90 days minimum)
- [ ] Included all available metrics:
  - [ ] Views, Watch Time, Subscribers
  - [ ] Engagement metrics (likes, comments, shares)
  - [ ] Traffic sources
  - [ ] Demographics
  - [ ] Device types

### Competitive Data
- [ ] Identified 3-5 direct competitors
- [ ] Gathered competitor metrics:
  - [ ] Upload frequency
  - [ ] Average views per video
  - [ ] Subscriber growth rates
  - [ ] Top performing content

### Content Library
- [ ] Listed all videos from audit period
- [ ] Categorized by:
  - [ ] Content type
  - [ ] Topic/niche
  - [ ] Upload date
  - [ ] Performance tier

---

## Phase 2: Content Performance Analysis

### Content Type Analysis
- [ ] Calculated average views per content type
- [ ] Calculated average watch time per content type
- [ ] Calculated engagement rate per content type
- [ ] Calculated subscriber conversion rate per content type
- [ ] Identified best-performing content type
- [ ] Identified worst-performing content type

### Topic Analysis
- [ ] Analyzed performance by topic
- [ ] Identified high-performing topics
- [ ] Identified underperforming topics
- [ ] Found topic performance patterns

### Top Performers
- [ ] Identified top 20% of videos
- [ ] Analyzed common characteristics
- [ ] Documented winning patterns

### Underperformers
- [ ] Identified bottom 20% of videos
- [ ] Analyzed common issues
- [ ] Documented losing patterns

---

## Phase 3: Engagement Analysis

### Click-Through Rate (CTR)
- [ ] Calculated CTR for each video
- [ ] Identified videos with CTR > 5% (excellent)
- [ ] Identified videos with CTR < 2% (needs improvement)
- [ ] Analyzed high-CTR thumbnail/title patterns
- [ ] Analyzed low-CTR thumbnail/title patterns

### Average View Duration (AVD)
- [ ] Calculated AVD percentage for each video
- [ ] Identified videos with AVD > 50% (excellent)
- [ ] Identified videos with AVD < 30% (needs improvement)
- [ ] Analyzed retention curves
- [ ] Identified drop-off points

### Engagement Rate
- [ ] Calculated engagement rate per video
- [ ] Identified high-engagement videos
- [ ] Identified low-engagement videos
- [ ] Analyzed engagement patterns

---

## Phase 4: Audience Growth Analysis

### Subscriber Growth
- [ ] Plotted subscriber growth over time
- [ ] Identified growth spikes
- [ ] Correlated spikes with content
- [ ] Calculated subscriber conversion rate
- [ ] Identified videos driving most subscribers

### View Velocity
- [ ] Tracked views in first 24 hours
- [ ] Tracked views in first 7 days
- [ ] Tracked views in first 30 days
- [ ] Identified videos with strong initial momentum
- [ ] Identified videos with long-tail growth

---

## Phase 5: Competitive Analysis

### Benchmarking
- [ ] Compared average views to competitors
- [ ] Analyzed subscriber growth rates
- [ ] Compared engagement rates
- [ ] Identified performance gaps

### Content Gap Analysis
- [ ] Listed top-performing videos from competitors
- [ ] Identified topics you haven't covered
- [ ] Found content formats you're missing
- [ ] Noted trending topics you're not addressing

### Trend Analysis
- [ ] Identified trending topics in your niche
- [ ] Checked which competitors adopted trends first
- [ ] Analyzed performance of trend-based content
- [ ] Identified opportunities for early trend adoption

---

## Phase 6: Technical Audit

### SEO Performance
- [ ] Analyzed titles of top-performing videos
- [ ] Checked keyword usage in titles
- [ ] Identified title patterns that work
- [ ] Analyzed description length and keywords
- [ ] Reviewed tag usage and effectiveness
- [ ] Compared top vs. bottom performers

### Thumbnail Audit
- [ ] Compared CTR by thumbnail style
- [ ] Identified visual elements that work
- [ ] Analyzed color schemes and text usage
- [ ] Reviewed thumbnail consistency

### Channel Organization
- [ ] Reviewed playlist organization
- [ ] Checked playlist performance
- [ ] Analyzed how playlists affect watch time
- [ ] Identified organization opportunities

---

## Phase 7: Opportunity Identification

### Content Opportunities
- [ ] Listed underserved topics in your niche
- [ ] Identified high-performing content types you're not using
- [ ] Found trending topics you can cover
- [ ] Discovered audience questions you can answer

### Optimization Opportunities
- [ ] Identified videos with high potential but low performance
- [ ] Found content that could benefit from re-optimization
- [ ] Discovered playlist opportunities
- [ ] Identified collaboration possibilities

### Risk Assessment
- [ ] Identified content types that consistently underperform
- [ ] Found topics that don't resonate
- [ ] Discovered technical issues affecting performance
- [ ] Identified engagement problems

---

## Phase 8: Action Planning

### Immediate Actions (Week 1)
- [ ] Optimize underperforming content (thumbnails, titles)
- [ ] Double down on winning content types
- [ ] Fix technical issues
- [ ] Improve playlist organization

### Short-Term Actions (Month 1)
- [ ] Adjust content strategy
- [ ] Implement engagement improvements
- [ ] Fill content gaps
- [ ] Adopt successful competitor strategies

### Long-Term Actions (Quarter 1)
- [ ] Evolve channel strategy
- [ ] Build content production systems
- [ ] Implement advanced growth strategies
- [ ] Explore new content formats
